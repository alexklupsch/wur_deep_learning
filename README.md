# wur_deep_learning
Project for Airborne Image Classification of the Deep Learning Course - Wageningen University 2026.

This project explores different deep learning models for the classification of airborne images from the UC Merced Dataset, both for single label as well as multi label classification. 
The project is divided into 4 jupyter notebooks, one each for a resnet/cnn as well as a vision transformer implementation of the different cases.

Hope you enjoy going through our code.
